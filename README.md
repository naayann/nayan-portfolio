# My Developer Portfolio

This is my personal developer portfolio built using React, Vite, and Tailwind CSS.
Designed to showcase projects, skills, and experience with a modern, responsive UI.

**Live Demo** https://nayansarania.netlify.app

### Tech Stack
- React
- Vite
- Tailwind CSS
- JavaScript

### Getting Started
```
npm install
npm run dev
```

### Deployment
The project is currently deployed on Netlify for development and preview purposes.
After production completion, it will be deployed using Vercel.
